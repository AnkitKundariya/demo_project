Demo Project for _`~~Techwondoe~~`_

Please Follow given steps to up and run this project.
1. Clone GIT Repo
2. create a virtual env of python v3.8
3. `cd furbaby` (make sure you are at the same level as manage.py file)
4. run this command to install required pip packages.
   1. `pip install -r requirements/local.txt`
5. activate virtual env
6. To Run project execute following command.
   1. `./manage.py runserver --settings=furbaby.local`
